# Интерактивная фотогалерея «Mesto». Проект разделен на несколько спринтов.
# 1. Валидация форм

## Описание проекта
Начальный этап разработки проекта «Mesto». Реализована логика работы всплывающих окон, форм и валидации на стороне клиента.

## Основной функционал
* Открытие и закрытие popup-окон.
* Редактирование профиля.
* Валидация форм.
* Добавление карточек.
* Удаление карточек.

## Используемые технологии
* HTML5
* CSS3
* JavaScript
* DOM API
* БЭМ

## Полученные навыки
* Работа с DOM.
* Написание валидации форм.
* Модульная организация JS-кода.


# 2. Работа с API

## Описание проекта
Версия проекта с серверной частью: получение данных пользователя и карточек, лайки, добавление карточек на сервер, редактирование информации профиля.

## Основной функционал
* Загрузка данных с API.
* Добавление карточек на сервер.
* Удаление карточек.
* Лайки и снятие лайков.
* Обновление аватарки.
* Синхронизация интерфейса с сервером.

## Используемые технологии
* JavaScript
* Fetch API
* Промисы
* Асинхронные операции

## Полученные навыки
* Работа с REST API.
* Обработка ошибок в интерфейсе.
* Асинхронное программирование.

# 4. ООП-архитектура

## Описание проекта

## Полученные навыки
